# Customer-Churn-Prediction-Business-Insights
End-to-end machine learning project to predict customer churn in a subscription-based business, combining data preprocessing, feature engineering, model evaluation, and business insights, with results prepared for visualization in a Power BI dashboard.
The project combines data preprocessing, feature engineering, and logistic regression, translating model results into actionable business insights and a Power BI–ready dataset.

## Key Topics
- Customer churn prediction
- Business-oriented model evaluation
- Risk-based customer segmentation
- Power BI dataset preparation

## Tools
Python, pandas, scikit-learn, matplotlib, seaborn, Power BI

## Files
- `01_customer_churn_prediction_business_insights.ipynb` – Full analysis and modeling
- `Churn_PowerBI_Dataset.csv` – Final dataset for Power BI
